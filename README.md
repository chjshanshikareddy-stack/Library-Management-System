# Library Management System

## Project Overview
This is a Library Management System developed using MySQL. The project demonstrates database design, SQL queries, joins, views, stored procedures, and triggers.

## Features
- Database creation
- Table creation
- Data insertion
- SELECT queries
- JOIN queries
- Aggregate functions
- UPDATE and DELETE operations
- Views
- Stored Procedures
- Triggers
- ER Diagram

## Database Tables
- Authors
- Books
- Members
- Borrow
