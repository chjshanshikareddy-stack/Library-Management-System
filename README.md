# Library Management System

## Project Overview

The Library Management System is a database project developed using MySQL Workbench. It helps manage books, authors, members, and book borrowing records. The project demonstrates the use of SQL commands and database concepts such as primary keys, foreign keys, joins, views, stored procedures, and triggers.

## Features

- Manage authors
- Manage books
- Manage members
- Record book borrowing details
- View borrowed books
- Generate reports using SQL queries
- Prevent invalid updates using triggers

## Technologies Used

- MySQL
- MySQL Workbench
- SQL

## Database Tables

1. Authors
2. Books
3. Members
4. Borrow
